Place your local 3D car files in this folder.

Default model paths used by the homepage hero:
- `/models/toyota-car.glb`
- `/models/hyundai-car.glb`
- `/models/bmw-car.glb`

Recommended:
- Use compressed `.glb` files for faster loading
- Keep filenames lowercase with hyphens
- Prefer files under 10 MB for a smoother homepage experience

You can also upload a model from the admin page when Firebase Storage is configured.
